#### 크기 단위
- em 
- rem (root em) : 최상위 요소의 사이즈 기준(보통은 16px)으로 배수 단위 가짐
- viewport : 디바이스 크기에 따라 달라지는 것 

#### 박스모델
모든 요소는 네모(박스모델) -> 좌상단에서 우하향으로 채워짐
위 아래 여백 공간이 겹치게 되면, 둘의 여백을 합치는 것이 아니라, 두 여백 값 중 큰 값을 margin 값으로 사용

#### CSS position
A
B
- absolute : 우리눈으로 볼 때는 위치가 그대로인 것처럼 보이지만, 공간이 사라짐으로써 B가 A의 뒤에 숨은 것처럼 보임
- relative : 원래 위치(공간)는 그대로 남겨두고, 내용물만 이동됨
=> relative로 상대 위치 잡아가며 처리할 때는, 본인의 공간을 그대로 두고 이동하는 것이기 때문에 normal flow에서 벗어나는 것이 아니다!! 다른 레이아웃에 크게 영향을 미치지는 않음
=> absolute는 변경되면서 기존의 공간이 없어지기 때문에, 다른 요소들의 모든 위치가 다 변할 수 있음
